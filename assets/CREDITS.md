# Crédits & licences des visuels

## Exercices — `assets/exo/`
Source : [free-exercise-db](https://github.com/yuhonas/free-exercise-db) (yuhonas).
Licence : **Unlicense** (domaine public) — aucune attribution requise.
25/27 exercices couverts. Manquants (pas d'image) : `hollow-hold`, `bird-dog`.
Photo unique par exercice (posture). Nom de fichier = id de l'exo dans `data/exercises.json`.

## Plats — `assets/recipes/`
Source : [TheMealDB](https://www.themealdb.com/) (API gratuite).
Usage personnel / éducatif. Images approximatives (match par mot-clé, pas exactes au plat).
11/12 plats couverts. Manquant : `wrap-poulet-avocat`.

## Remplacer / compléter
Dépose un fichier `<id>.jpg` (ou `.webp`) dans le bon dossier pour ajouter/écraser un visuel.
Surcharge possible via les champs `image`/`anim` (exos) ou `image` (recettes) dans les JSON.
